# OWASP Juice Shop Notes
This repository contains my learning notes and write-ups 
while practicing web application security using OWASP Juice Shop.

The goal of this project is to understand common web vulnerabilities and 
how they can be exploited and mitigated.

# Purpose
- identify and explain vulnerabilities exist in it
- strengthen web application security fundamentals
